What is NPP-PRO ?

NPP-PRO is Syntax highlighting and Auto Completion for .PRO files in Notepad++

What is .PRO Files ?

.PRO Files is
Creo config file

PTC Creo, formerly known as Pro/ENGINEER is a parametric, integrated 3D CAD/CAM/CAE solution created by Parametric Technology Corporation (PTC). It was the first to market[2] with parametric, feature-based, associative solid modeling software.

Syntax Highlighting
INSTALLATION

  • ¤ Open Notepad++.
  • ¤ Click in the menu on: "Language -> Define your own Language -> Import".
  • ¤ Import the file "syntaxhighlighting/PRO.xml".
  • ¤ Restart Notepad++.

Auto Completion
INSTALLATION

The steps in the section "Syntax Highlighting" are required for auto completion.

  • ¤ Copy the file "autocompletion/PRO.xml" into the folder "C:\Program Files (x86)\Notepad++\plugins\APIs" (or where you installed Notepad++).
  • ¤ Restart Notepad++.
  • ¤ Open "Settings -> Preferences..." in Notepad++.
  • ¤ Go to the tab "Auto-Completion".
  • ¤ Choose following options:
  • ----¤ Enable "Enable auto-completion on each input"
  • ----¤ Choose "Function completion"
  • ----¤ Set "From 3th character" (or what you prefer)
